# <a name="virtualdirectoryreportserver-property-wmi-msreportserver_configurationsetting"></a>Proprietà VirtualDirectoryReportServer (MSReportServer_ConfigurationSetting WMI)
  Restituisce la directory virtuale per l'applicazione del server di report. Di sola lettura.  
  
## <a name="syntax"></a>Sintassi  
  
```vb  
Public Dim VirtualDirectoryReportServer As String  
```  
  
```csharp  
public string VirtualDirectoryReportServer;  
```  
  
## <a name="property-values"></a>Valori della proprietà  
 Oggetto `string` che contiene la directory virtuale.
