# <a name="installation-prerequisites"></a>Prerequisiti di installazione
  Per la corretta installazione di [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] , Ã¨ necessario che nel computer di installazione siano disponibili i componenti software riportati di seguito. Tali componenti verranno installati dal programma di installazione di [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] :  
  
-   [!INCLUDE[dnprdnshort](../../includes/dnprdnshort-md.md)] versione 4.0 per tutte le altre edizioni di [!INCLUDE[ssCurrent](../../includes/sscurrent-md.md)]  
  
-   [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] File di supporto per l'installazione  
  
-   [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] Native Client  
  
> [!NOTE]  
>  Questi componenti non vengono rimossi se l'installazione viene annullata o non viene completata. Per rimuoverli, usare **Installazione applicazioni** nel Pannello di controllo.  
  
## <a name="options"></a>Opzioni  
 Per installare i file necessari, fare clic su **Installa**. Per uscire dall'installazione, scegliere **Annulla**.
