# <a name="propertyvaluetype-property-sqlserviceadvancedproperty-class"></a>Proprietà PropertyValueType (classe SqlServiceAdvancedProperty)
  Ottiene il tipo di dati di una proprietà.  
  
## <a name="syntax"></a>Sintassi  
  
```  
  
object  
.PropertyValueType [= value]  
```  
  
## <a name="parts"></a>Parti  
 *object*  
 Oggetto della [classe SqlServiceAdvancedProperty](sqlserviceadvancedproperty-class.md) che rappresenta una proprietà avanzata.  
  
## <a name="property-valuereturn-value"></a>Valore proprietà/Valore restituito  
 Valore `uint32` che rappresenta il tipo di dati della proprietà.
