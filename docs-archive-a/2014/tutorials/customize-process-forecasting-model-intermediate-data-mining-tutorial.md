## <a name="setting-time-series-parameters"></a><span data-ttu-id="1ac99-110">Impostazione dei parametri di Time Series</span><span class="sxs-lookup"><span data-stu-id="1ac99-110">Setting Time Series Parameters</span></span>  
 <span data-ttu-id="1ac99-111">**Hint di periodicità**</span><span class="sxs-lookup"><span data-stu-id="1ac99-111">**Periodicity Hints**</span></span>  
  
 <span data-ttu-id="1ac99-112">Il *PERIODICITY_HINT* parametro fornisce all'algoritmo informazioni sui periodi di tempo aggiuntivi che si prevede di visualizzare nei dati.</span><span class="sxs-lookup"><span data-stu-id="1ac99-112">The *PERIODICITY_HINT* parameter provides the algorithm with information about additional time periods that you expect to see in the data.</span></span> <span data-ttu-id="1ac99-113">Per impostazione predefinita, i modelli Time Series tenteranno di rilevare automaticamente un modello nei dati.</span><span class="sxs-lookup"><span data-stu-id="1ac99-113">By default, time series models will try to automatically detect a pattern in the data.</span></span> <span data-ttu-id="1ac99-114">Se tuttavia si conosce già il ciclo temporale previsto, l'indicazione di un hint di periodicità può migliorare potenzialmente l'accuratezza del modello.</span><span class="sxs-lookup"><span data-stu-id="1ac99-114">However, if you already know the expected time cycle, providing a periodicity hint can potentially improve the accuracy of the model.</span></span> <span data-ttu-id="1ac99-115">Se si fornisce tuttavia l'hint di periodicità errato, l'accuratezza può diminuire. Di conseguenza, in caso di dubbi sul valore da utilizzare, è preferibile utilizzare il valore predefinito.</span><span class="sxs-lookup"><span data-stu-id="1ac99-115">However, if you provide the wrong periodicity hint, it can decrease accuracy; therefore, if you are not sure what value should be used, it is best to use the default.</span></span>  
  
 <span data-ttu-id="1ac99-116">Ad esempio, la vista utilizzata per questo modello aggrega dati di vendita da [!INCLUDE[ssAWDWsp](../includes/ssawdwsp-md.md)] su base mensile.</span><span class="sxs-lookup"><span data-stu-id="1ac99-116">For example, the view used for this model aggregates sales data from [!INCLUDE[ssAWDWsp](../includes/ssawdwsp-md.md)] on a monthly basis.</span></span> <span data-ttu-id="1ac99-117">Ogni intervallo di tempo utilizzato dal modello rappresenta pertanto un mese e tutte le stime saranno anch'esse indicate in termini di mesi.</span><span class="sxs-lookup"><span data-stu-id="1ac99-117">Therefore each time slice used by the model represents one month, and all predictions will also be in terms of months.</span></span> <span data-ttu-id="1ac99-118">Poiché sono presenti 12 mesi in un anno e si prevede che i modelli di vendita vengano ripetuti più o meno su base annuale, impostare il parametro *PERIODICITY_HINT* su per `12` indicare che 12 intervalli di tempo (mesi) costituiscono un ciclo di vendita completo.</span><span class="sxs-lookup"><span data-stu-id="1ac99-118">Since there are 12 months in a year and you expect that sales patterns more or less repeat on a yearly basis, you will set the *PERIODICITY_HINT* parameter to `12`, to indicate that 12 time slices (months) constitute one complete sales cycle.</span></span>  
  
 <span data-ttu-id="1ac99-119">**Metodo di previsione**</span><span class="sxs-lookup"><span data-stu-id="1ac99-119">**Forecasting Method**</span></span>  
  
 <span data-ttu-id="1ac99-120">Il parametro *FORECAST_METHOD* controlla se l'algoritmo Time Series è ottimizzato per le stime a breve o a lungo termine.</span><span class="sxs-lookup"><span data-stu-id="1ac99-120">The *FORECAST_METHOD* parameter controls whether the time series algorithm is optimized for short-term or long-term predictions.</span></span> <span data-ttu-id="1ac99-121">Per impostazione predefinita, il parametro *FORECAST_METHOD* è impostato su Mixed, il che significa che due algoritmi diversi vengono combinati e bilanciati per fornire risultati validi sia per la stima a breve termine che per quella a lungo termine.</span><span class="sxs-lookup"><span data-stu-id="1ac99-121">By default, the *FORECAST_METHOD* parameter is set to MIXED, which means that two different algorithms are blended and balanced to provide good results for both short-term and long-term prediction.</span></span>  
  
 <span data-ttu-id="1ac99-122">Se tuttavia si desidera utilizzare un algoritmo particolare, è possibile modificare il valore in ARIMA o ARTXP.</span><span class="sxs-lookup"><span data-stu-id="1ac99-122">However, if you know that you want to use a particular algorithm, you can change the value to either ARIMA or ARTXP.</span></span>  
  
 <span data-ttu-id="1ac99-123">**Ponderazione delle stime a lungo termine e a breve termine**</span><span class="sxs-lookup"><span data-stu-id="1ac99-123">**Weighting Long-Term vs. Short-Term Predictions**</span></span>
