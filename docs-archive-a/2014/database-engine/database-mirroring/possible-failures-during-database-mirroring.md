## <a name="failures-due-to-soft-errors"></a><span data-ttu-id="148f5-143">Errori provocati da errori software</span><span class="sxs-lookup"><span data-stu-id="148f5-143">Failures Due to Soft Errors</span></span>

### <a name="the-mirroring-time-out-mechanism"></a><span data-ttu-id="148f5-150">Meccanismo di timeout per il mirroring</span><span class="sxs-lookup"><span data-stu-id="148f5-150">The Mirroring Time-Out Mechanism</span></span>

 <span data-ttu-id="148f5-153">Per mantenere aperta una connessione, è necessario che un'istanza del server riceva un ping su tale connessione entro il periodo definito dalla somma del valore del timeout e del tempo necessario per l'invio del ping successivo.</span><span class="sxs-lookup"><span data-stu-id="148f5-153">To keep a connection open, a server instance must receive a ping on that connection in the time-out period defined, plus the time that is required to send one more ping.</span></span> <span data-ttu-id="148f5-154">La ricezione di un ping durante il periodo di timeout indica che la connessione è ancora aperta e che le istanze del server comunicano attraverso tale connessione.</span><span class="sxs-lookup"><span data-stu-id="148f5-154">Receiving a ping during the time-out period indicates that the connection is still open and that the server instances are communicating over it.</span></span> <span data-ttu-id="148f5-155">Quando riceve un ping, un'istanza del server reimposta il contatore del timeout su tale connessione.</span><span class="sxs-lookup"><span data-stu-id="148f5-155">On receiving a ping, a server instance resets its time-out counter on that connection.</span></span>

 <span data-ttu-id="148f5-164">**Per modificare il valore di timeout (solo modalità a protezione elevata)**</span><span class="sxs-lookup"><span data-stu-id="148f5-164">**To change the time-out value (high-safety mode only)**</span></span>
