## <a name="adding-the-target-to-a-session"></a><span data-ttu-id="e1382-145">Aggiunta della destinazione a una sessione</span><span class="sxs-lookup"><span data-stu-id="e1382-145">Adding the Target to a Session</span></span>  
 <span data-ttu-id="e1382-146">Per aggiungere la destinazione file dell'evento a una sessione di eventi estesi, è necessario includere le istruzioni seguenti quando si crea o si modifica una sessione eventi, sostituendo *file_name* con il nome e il percorso del file desiderati:</span><span class="sxs-lookup"><span data-stu-id="e1382-146">To add the event file target to an Extended Events session, you would include the following statements when you create or alter an event session, replacing *file_name* with the desired file name and path:</span></span>
